考え方の問題

初めに

  • 考え方を見るためであり正解はありません
  • 問題に対する質問は自由
  • 考えは背景が緑の黒い枠で囲まれたエリアにHTML中に直接記述してください

「インターネットが見れないんですけど」と問い合わせがありました。問題を解決するのにどのような意図でどのような質問をしていきますか？

2.応答が返ってこない

大量にDBに保存されている売上データに対し集計を掛けましたがいくら経っても応答が返ってきません。何をどのように探り解決していきますか？

プログラム問題

初めに

  • 回答で作ったプログラムはtest.jsもしくは新しく任意にファイルを作成し記述し、このHTMLファイルから呼び出されるようにすること
  • 作ったプログラムの出力は、背景が緑の黒い枠で囲まれたエリアに出力すること(4の問題は除く)
  • Chromeの最新で動作すればOKとする
  • ES6やTypeScriptで記述してもOKとする
  • 別のライブラリなどを組み込んで使用してもOKとする
  • 良いと思った仕様を追加してもOKとする
  • 検索してもOKだが、1の問題は出来ないことが開発者になってはいけないレベルで恥ずかしいのでプログラムの文法以外は自力で考えることを推奨
  • 問題に対する質問は自由

1.FizzBuzz問題

1から100までの数をプリントするプログラムを書け。ただし3の倍数のときは数の代わりに「Fizz」と、5の倍数のときは「Buzz」とプリントし、3と5両方の倍数の場合には「FizzBuzz」とプリントすること。

2.金種計算問題

金額を引数に渡したら、金種を計算し、それぞれの枚数を出力するプログラムを書け。使用する金種は10000円、5000円、1000円、500円、100円、50円、10円、5円、1円の9種とする。

出力例

19769 => 1=4,5=1,10=1,50=1,100=2,500=1,1000=4,5000=1,10000=1
1234 => 1=4,5=0,10=3,50=0,100=2,500=0,1000=1,5000=0,10000=0
9281 => 1=1,5=0,10=3,50=1,100=2,500=0,1000=4,5000=1,10000=0
